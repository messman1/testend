# 청소년 커뮤니티 앱

중학생들이 시험 후 건전하게 놀 수 있는 공간과 놀이를 발견하고, 친구들과 모임을 만드는 커뮤니티 플랫폼

## 🚀 시작하기

### 설치

```bash
cd teen-community-app
npm install
```

### 개발 서버 실행

```bash
npm run dev
```

브라우저에서 `http://localhost:3000` 을 열어주세요.

### 빌드

```bash
npm run build
```

## 📱 주요 기능

- **홈**: 인기 카테고리 및 빠른 액션
- **탐색**: 장소 검색 및 카테고리별 필터링
- **추천**: "오늘 뭐하지?" 맞춤 추천
- **모임**: 친구들과 모임 만들기 및 관리
- **소식**: 커뮤니티 피드 (신규 오픈, 후기, 이벤트)
- **프로필**: 내 활동 통계 및 뱃지

## 🛠 기술 스택

- React 19.2.3
- Vite 7.2.7
- React Router DOM 7.10.1
