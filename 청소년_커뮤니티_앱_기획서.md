# 청소년 커뮤니티 앱 기획서

## 📱 앱 개요
**목표**: 중학생들이 시험 후 건전하게 놀 수 있는 공간과 놀이를 발견하고, 친구들과 모임을 만드는 커뮤니티 플랫폼

**타겟**: 중학생 (13-16세)

**플랫폼**: 모바일 앱 (iOS/Android - React Native)

---

## 🎯 핵심 기능

### 1. 놀이 장소 추천 및 검색

#### 1-1. 상업 공간 (우선순위 높음)
청소년들이 자주 가는 인기 장소를 카테고리별로 추천

**카테고리:**
- 🎬 **영화관**: 최신 개봉작, 청소년 할인 정보
- 🎯 **체험 엔터테인먼트**: 방탈출 카페, VR 카페, 보드게임방
- 🎤 **노래방**: 코인노래방, 일반 노래방
- 🍜 **먹거리**: 떡볶이, 마라탕, 치킨, 햄버거, 디저트 카페
- 🎮 **오락실/PC방**: 게임 즐기기 좋은 곳
- 📚 **북카페/스터디카페**: 조용히 수다 떨거나 공부할 수 있는 곳

**기능:**
- 위치 기반 주변 장소 검색
- 가격대 필터 (용돈으로 갈 수 있는 곳)
- 친구들 평점 및 후기
- 예산 계산기 (1인당 비용 × 인원)
- 영업시간 및 청소년 이용 가능 시간 표시

#### 1-2. 공공 공간 (보조)
- 📖 **공공도서관**: 스터디룸, 휴게 공간 정보
- 🏛️ **청소년센터**: 무료/저렴한 프로그램 소개
- 🏞️ **공원/야외 공간**: 산책, 운동하기 좋은 곳

### 2. 놀이 제안 - "시험 끝났는데 뭐하지?"

시간과 기회는 있는데 무엇을 할지 정하지 못한 친구들을 위한 맞춤 추천

#### 2-1. 상황별 추천
**질문 기반 추천 시스템:**
- "우리 몇 명이야?" → 2명 / 3-5명 / 6명 이상
- "예산은 얼마야?" → 1만원 이하 / 1-2만원 / 2만원 이상
- "뭐 하고 싶어?" → 먹기 / 놀기 / 쉬기 / 활동적인 거
- "시간은 얼마나 있어?" → 1-2시간 / 반나절 / 하루종일

**추천 예시:**
```
✨ 너희에게 딱!
👥 3명 | 💰 1.5만원 | ⏱️ 3시간

1. 방탈출 카페 → 떡볶이 → 코인노래방
   예상 비용: 1인 13,000원

2. 보드게임방 3시간 패키지
   예상 비용: 1인 12,000원
```

#### 2-2. 인기 코스
- 다른 친구들이 많이 간 코스 공유
- "이번 주 인기 코스 TOP 10"
- 계절별/날씨별 추천 (비 오는 날, 더운 날 등)

#### 2-3. 랜덤 추천 (룰렛)
- 정말 못 정하겠을 때 랜덤으로 뽑아주기
- "오늘의 운명 놀이" 기능

---

### 3. 모임 만들기

#### 3-1. 모임 생성
- 모임 제목 및 설명
- 날짜/시간 설정
- 장소 선택 (앱 내 추천 장소에서 선택 가능)
- 최대 인원 설정
- 공개/비공개 설정

#### 3-2. 모임 참여
- 친구 초대 (친구 목록 또는 링크 공유)
- 참여 신청 및 수락
- 참석 여부 체크
- 채팅 기능 (모임별 단체 채팅방)

#### 3-3. 모임 관리
- 일정 변경 알림
- 참가자 관리
- 비용 더치페이 계산기
- 모임 후기 및 사진 공유

---

### 4. 소식 및 정보 공유

#### 4-1. 우리 동네 새로운 곳
- "신규 오픈" 태그로 새로 생긴 가게 소개
- 예: "🆕 강남역에 보드게임 카페 새로 생겼어요!"
- 오픈 이벤트, 할인 정보

#### 4-2. 이벤트 및 할인 정보
- 청소년 할인 이벤트
- 시즌별 프로모션 (방학 특가, 시험 기간 후 이벤트 등)
- 무료 체험 기회

#### 4-3. 커뮤니티 피드
- 친구들이 다녀온 장소 후기
- "여기 완전 재밌어!" 추천 글
- 사진/영상 공유 (먹은 음식, 재밌는 순간)

#### 4-4. 지역별 게시판
- 우리 동네 친구들과 정보 교류
- "○○동 청소년 게시판"

## 🎨 주요 화면 구성

### 메인 화면 (홈)
```
[상단]
- 위치 설정 (우리 동네)
- 검색창

[중단]
- "오늘 뭐하지?" 빠른 추천 버튼
- 인기 카테고리 (영화관, 방탈출, 먹거리...)
- 이번 주 인기 장소 TOP 5

[하단]
- 친구들 최근 활동
- 우리 동네 새 소식
```

### 놀이 추천 화면
```
[질문 시작]
"시험 끝났는데 뭐하지?"

[단계별 선택]
→ 인원 수 선택
→ 예산 선택
→ 활동 유형 선택
→ 시간 선택

[결과]
→ 맞춤 코스 3가지 추천
→ 각 코스별 상세 정보
→ "이 코스로 모임 만들기" 버튼
```

### 장소 상세 화면
```
[이미지 갤러리]
[기본 정보]
- 카테고리, 가격대, 위치
- 영업시간
- 청소년 이용 가능 시간

[친구들 평가]
- 별점 및 후기
- "재미있어요" "맛있어요" 등 태그

[버튼]
- 모임 만들기
- 찜하기
- 공유하기
```

### 모임 화면
```
[탭]
- 내 모임
- 초대받은 모임
- 지난 모임

[모임 카드]
- 모임 제목
- 날짜/시간/장소
- 참여 인원
- 채팅 바로가기
```

### 커뮤니티 화면
```
[필터]
- 전체 / 우리 동네 / 친구들만

[피드]
- 신규 오픈 소식
- 친구들 후기
- 이벤트 정보
```

## 🔧 기술 스택 제안 (React Native 기준)

### Frontend
- **Framework**: React Native
- **Navigation**: React Navigation
- **State Management**: Redux Toolkit 또는 Zustand
- **UI Components**: React Native Paper 또는 NativeBase
- **Maps**: React Native Maps (장소 표시)
- **Location**: React Native Geolocation

### Backend (제안)
- **API**: Node.js + Express 또는 Firebase
- **Database**: Firebase Firestore 또는 MongoDB
- **Authentication**: Firebase Auth 또는 JWT
- **File Storage**: Firebase Storage (이미지 업로드)
- **Push Notification**: Firebase Cloud Messaging

### 외부 API
- 카카오맵 또는 네이버 지도 API (장소 정보)
- 영화진흥위원회 API (영화 정보)

## 🎯 개발 우선순위

### Phase 1 (MVP - 최소 기능)
1. 장소 검색 및 카테고리별 보기
2. 장소 상세 정보 및 후기
3. 간단한 놀이 추천 (질문 기반)
4. 모임 만들기 및 참여
5. 기본 프로필 및 친구 기능

### Phase 2 (확장)
1. 커뮤니티 피드 (소식 공유)
2. 찜하기 및 히스토리
3. 위치 기반 주변 장소 추천
4. 더치페이 계산기
5. 고급 필터링 (가격, 거리, 평점)

### Phase 3 (고도화)
1. 놀이 코스 추천 알고리즘 개선
2. 포인트/뱃지 시스템
3. 실시간 인기 장소 랭킹
4. 이벤트/할인 정보 자동 수집
5. AR 기능 (주변 장소 찾기)

## 📝 고려사항

### 청소년 타겟의 특징
- **간편함**: 복잡한 기능보다는 직관적이고 빠른 UX
- **트렌드**: 요즘 유행하는 디자인과 기능 (숏폼 비디오, 스티커 등)
- **소셜**: 친구들과 공유하고 소통하는 기능 중요
- **비용 민감도**: 가격 정보와 할인 정보에 높은 관심

### 데이터 수집 방안
- 초기: 수동으로 지역별 인기 장소 등록
- 중기: 사용자들이 직접 장소 등록 및 추천
- 장기: API 연동 및 자동 크롤링

### 수익 모델 (선택사항)
- 제휴 광고 (장소 홍보)
- 신규 오픈 매장 홍보 슬롯
- 프리미엄 기능 (무제한 찜, 우선 추천 등)
